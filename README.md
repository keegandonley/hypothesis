# hypothesis.sh

A collection of small, focused developer tools. No accounts, no tracking — just utilities that work.

## Tools

| Tool                                                     | Description                                                                | Docs                                               |
| -------------------------------------------------------- | -------------------------------------------------------------------------- | -------------------------------------------------- |
| [webhook](https://hypothesis.sh/webhook)                 | Inspect incoming HTTP requests in real time with a personal webhook URL    | [docs](https://hypothesis.sh/docs/webhook)         |
| [base64](https://hypothesis.sh/base64)                   | Encode and decode base64 strings with live sync and shareable permalinks   | [docs](https://hypothesis.sh/docs/base64)          |
| [uuid](https://hypothesis.sh/uuid)                       | Generate UUIDs of any version (v1, v4, v7) with one click                  | [docs](https://hypothesis.sh/docs/uuid)            |
| [regex](https://hypothesis.sh/regex)                     | Test regular expressions against sample input with live match highlighting | [docs](https://hypothesis.sh/docs/regex)           |
| [datetime](https://hypothesis.sh/datetime)               | Parse, convert, and format dates and timestamps                            | [docs](https://hypothesis.sh/docs/datetime)        |
| [pretty-print](https://hypothesis.sh/pretty-print)       | Format and syntax-highlight JSON                                           | [docs](https://hypothesis.sh/docs/pretty-print)    |
| [urlencode](https://hypothesis.sh/urlencode)             | URL-encode and decode strings                                              | [docs](https://hypothesis.sh/docs/urlencode)       |
| [iframe-proxy](https://hypothesis.sh/iframe-proxy)       | Proxy iframes with bidirectional postMessage relay and debug panel         | [docs](https://hypothesis.sh/docs/iframe-proxy)    |
| [message-stream](https://hypothesis.sh/message-stream)   | Capture and inspect postMessage events in real time                        | [docs](https://hypothesis.sh/docs/message-stream)  |
| [message-factory](https://hypothesis.sh/message-factory) | Build and send postMessage payloads interactively                          | [docs](https://hypothesis.sh/docs/message-factory) |

## Stack

Next.js · TypeScript · Planetscale (Postgres) · Vercel
