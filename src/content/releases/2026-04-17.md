# Spring Tools Update

## JSON Diff

Paste two JSON objects and see exactly what changed — additions, removals, and modifications highlighted side-by-side. Handles deeply nested structures and invalid JSON gracefully. Diffs are permalink-shareable.

## Color Shades

Generate a full tint/shade scale from any base color. Input hex, RGB, or HSL and get an 11-step scale (50 through 950) ready to copy as CSS custom properties. Preview each step on light and dark backgrounds.

## Basic Auth

Encode and decode HTTP Basic Authentication headers. Enter a username and password to get the `Authorization: Basic …` header value, or paste an existing header to decode it.

## Keycode Inspector

Press any key and see its `keyCode`, `key`, `code`, `which`, and modifier state.

## Markdown Preview

Live markdown editor and preview side-by-side.

## SQL Formatter

Paste any SQL query and get it back formatted with consistent indentation, keyword casing, and line breaks.

## SVG to JSX

Paste an SVG and get back a React-compatible JSX component with `camelCase` attribute names. Copy directly into a component file.

## OG Image Tester

Preview how a URL's Open Graph metadata renders as a social card — fetches `og:title`, `og:description`, and `og:image` and displays the result.
