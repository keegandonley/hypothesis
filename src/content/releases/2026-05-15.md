# Mobile App Version 1.3, New Reference Guides, and QR Code Decoder

## Mobile App — Version 1.3

**QR Code Decoder**

The app now includes a QR decoder tool. Point the camera at any QR code to decode its value without visiting the URL. You can also share a saved image from your photo library to decode a QR code from it. Previously scanned codes are saved to a history list for easy reference.

**Pinnable Tabs**

Tools and experiments can now be pinned to a dedicated Pinned tab for quick access using the pin icon on each card. This release also improves the reliability of saved tabs.

This version also includes minor bug fixes, improvements, and adjustments to the layout to enhance usability.

## Web — New Reference Guides

Four new reference guides are now available:

- **macOS Shortcuts** — keyboard shortcuts for macOS system actions
- **tmux** — key bindings and commands for tmux sessions
- **PostgreSQL Operators** — JSONB and array operators with examples
- **psql Meta-Commands** — backslash commands for the psql interactive terminal

## Web — Improvements and Fixes

- Updated the iframe proxy experiment with refreshed styling and improved event logging
- Fixed data inaccuracies in the HTTP headers, MIME types, and Unicode blocks references
- Updated documentation for the Base64, message stream, and webhook tools
