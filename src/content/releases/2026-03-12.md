# Core Tools Launch

- **Number Base Converter** — convert integers between binary, octal, decimal, and hexadecimal
- **HTML Entity Encoder** — encode and decode HTML entities (`&amp;`, `&#60;`, etc.)
- **Bitwise Calculator** — visualize AND, OR, XOR, NOT, and shift operations on 8, 16, and 32-bit integers
- **Hash Generator** — compute MD5, SHA-1, SHA-256, and SHA-512 hashes of any input string
- **JWT Inspector** — decode and inspect JWT headers and payloads without verification; supports HS256 and RS256 display
- **chmod Calculator** — build Unix file permission values visually and see the octal and symbolic representations
- **CIDR Calculator** — compute network address, broadcast, subnet mask, and host range from any CIDR block
- **Cron Expression Parser** — describe a cron expression in plain English and preview the next several scheduled times
- **Color Picker** — convert between hex, RGB, HSL, and HSV with a visual picker and one-click copy for each format
- **Lorem Ipsum Generator** — generate placeholder text by word, sentence, or paragraph count
- **Text Statistics** — character count, word count, sentence count, reading time, and Flesch readability score
- **QR Code Generator** — generate a QR code from any URL or text, downloadable as PNG
- **CSS Unit Converter** — convert between `px`, `rem`, `em`, `vw`, `vh`, `pt`, `pc`, and `cm` with configurable base font size and viewport dimensions
- **Date & Time** — format timestamps, convert between timezones, and compute date arithmetic
- **Unicode Explorer** — look up any Unicode character by name, code point, or paste — shows category, script, block, and HTML entity
