# Media & Screenshot Tools

## App Store Screenshot Generator

Compose a device frame, background gradient, and headline text and export at the correct pixel dimensions for each App Store slot.

## Audio Streaming

Connect to any HLS audio stream or standard audio URL and monitor playback state, buffering, and errors in real time.

## Video Streaming

Paste any `.m3u8` or standard video URL and play it back in the browser with fullscreen support and stream metadata display.

## Screen Capture

Capture your screen or a specific browser tab using the Screen Capture API and preview the stream directly in the browser.

## Media File Reference

Common media file extensions grouped by type — video, audio, image, document, and archive — with MIME types.
