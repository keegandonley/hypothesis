# Mobile App Version 1.2.1, Reference Improvements, and Homepage Filtering

## Mobile App — Version 1.2.1

**Reference display improvements**

Several reference tables now show richer detail rows. Unicode Blocks displays the assigned character count and code point range; ASCII shows decimal, hex, and octal codes alongside the abbreviation; Regex Syntax shows example patterns with their output; and Unix Signals shows a catchable/not-catchable badge.

**Deep linking**

The app now supports deep links, allowing external sources (notifications, web links) to open directly to specific screens.

**Smarter notification navigation**

Tapping a push notification now navigates directly to the Notifications sub-tab in the Events screen. If webhooks are disabled in settings, the app defaults to the Notifications tab on launch.

## Web — Reference and Homepage Updates

- Fixed the References API so reference tools resolve correctly from the tools list
- Homepage tool listing now supports filtering
- Various homepage layout and style refinements
