# ASCII Art, Scratch Pad & Timezones

## ASCII Art Generator

Convert any text into ASCII art using a selection of classic figlet-style fonts. Copy the result for use in terminal output, code comments, or READMEs.

## Scratch Pad

A minimal persistent text editor that saves to `localStorage`. Content survives page refreshes.

## Timezones Reference

Searchable table of IANA timezone identifiers with current UTC offsets and example city names.
