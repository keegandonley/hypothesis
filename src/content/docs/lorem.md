# lorem ipsum

Generate lorem ipsum placeholder text by words, sentences, or paragraphs — instantly, in your browser.

## Overview

Select a unit type and a count, then click **Regenerate** to produce new random text. The word bank is the classic lorem ipsum vocabulary drawn from Cicero's *de Finibus Bonorum et Malorum*.

## Unit types

| Type | Description |
|------|-------------|
| Words | N individual words picked at random |
| Sentences | N sentences of 8–15 words each, capitalized with a period |
| Paragraphs | N paragraphs of 3–6 sentences each, separated by blank lines |

## Controls

- **Unit** — toggle between Words, Sentences, and Paragraphs
- **Count** — number input or range slider (1–100)
- **Regenerate** — produce new random output for the current settings
- **Copy** — copy the full output to your clipboard

## URL parameters

`?type=<words|sentences|paragraphs>&count=<n>` — persists the selected type and count in the URL. The generated text itself is not in the URL (it changes on every regenerate).
